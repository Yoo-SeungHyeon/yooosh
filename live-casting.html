라이브캐스팅 - MSA 기반 인터랙티브 방송 플랫폼

Spring Boot React MSA Kubernetes Kafka Redis WebSocket
📋 Status: Planning & Design Phase

프로젝트 개요

크리에이터가 팬들과 실시간으로 소통하며 자신의 상품을 판매하고, 특별 이벤트를 통해 팬덤을 강화할 수 있는 MSA 기반의 인터랙티브 방송 플랫폼입니다. 실시간 채팅, 좋아요, 구독, 선착순 이벤트 등의 다양한 기능을 통해 크리에이터와 팬들이 더욱 밀접하게 연결될 수 있도록 설계된 혁신적인 플랫폼입니다.

실시간 소통

라이브 채팅과 실시간 반응으로 크리에이터와 팬 간의 즉석 소통

라이브 커머스

방송 중 직접 상품을 판매하고 구매할 수 있는 통합 쇼핑 시스템

특별 이벤트

선착순 한정판 굿즈 판매 등 팬덤 강화 이벤트

주요 활동
  • • 라이브 방송 관리
  • • 실시간 커뮤니케이션
  • • 성과 지표 분석
  • • 프로모션 이벤트 운영

주요 활동
  • • 고화질 스트리밍 시청

  • • 구독 및 알림 설정
  • • 이벤트 참여

주요 활동

핵심 기능

❤️

실시간 '좋아요' 기능

사용자는 라이브 방송 또는 VOD 영상에 '좋아요'를 누를 수 있으며, 좋아요 수는 실시간으로 집계되어 표시됩니다.

React UI Spring Boot Kafka Events Redis Counter
🔔

크리에이터 '구독' 기능

사용자는 특정 크리에이터 채널을 구독할 수 있으며, 구독 발생 시 크리에이터에게 즉시 알림이 전송됩니다.

MySQL 관계 Kafka Events 알림 서비스 트랜잭션
💬

실시간 채팅 기능

라이브 방송 중 모든 참여자는 WebSocket을 통해 실시간으로 메시지를 주고받을 수 있습니다.

WebSocket Kafka Queue Cassandra Chat Service
🏃‍♂️

선착순 이벤트 처리

크리에이터가 라이브 중 "선착순 N명" 이벤트를 시작할 수 있으며, 수만 명의 동시 요청에도 정확하게 선착순 N명만 처리됩니다.

Redis INCR Atomic Operations 동시성 해결 Event Service
