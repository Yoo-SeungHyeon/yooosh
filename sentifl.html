리팩토링 계획

졸업 작품으로 개발된 BGM 생성 AI 서비스를 현대적인 기술 스택과 아키텍처로 재구성하여 확장성과 유지보수성을 향상시키는 프로젝트입니다.

개선 목표

  • • 레거시 코드 현대화
  • • AI 모델 성능 최적화
  • • 사용자 경험 개선
  • • 클라우드 네이티브 전환

핵심 기능

🎼

AI 음악 생성

텍스트 설명이나 감정 키워드를 입력하면 딥러닝 모델이 그에 맞는 BGM을 자동 생성

Keras RNN/LSTM MIDI Generation
😊

감정 분석 엔진

텍스트의 감정을 분석하여 기쁨, 슬픔, 평온 등의 감정에 맞는 음악 스타일 결정

NLP Sentiment Analysis FastAPI
📝

인터랙티브 블로그

생성된 BGM과 함께 블로그 포스트 작성, 음악 재생이 가능한 리치 에디터 제공

React Rich Editor Audio Player
🎵

개인 음악 라이브러리

생성된 음악들을 저장하고 관리할 수 있는 개인 라이브러리, 태그 및 분류 기능
